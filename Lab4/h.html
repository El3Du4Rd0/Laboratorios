¿Qué diferencias y semejanzas hay entre Java y JavaScript?

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
getDate() que sirve para obtener el día del més, getMonth() que sirve para obtener el més, getFullYear() que sirve para obtener el año, getDay() que sirve para obtener el día y finalmente getTime() que regresa el valor numerico a la fecha y hora
¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
push() que agrega elementos al arreglo, pop() que elimina elementos en un arreglo, splice() que permite eliminar un elemento en una direccion en en especifico, indexOf() que regresa la posicion de un elemento en especifico y finalmente forEach() que ejecuta una funcion por cada elemento en el arreglo
¿Cómo se declara una variable con alcance local dentro de una función?
en general se puede hacer unicamente definiendo las variables con var, let o const segun las necesidades
¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
en general son malas practicas ya que contaminan el espacio en general de las demas funciones, dificulta el trabajo y la redaccion del documento y en general malas cosas.
